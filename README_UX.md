# Gorilla Jungle — UI/UX front prototype

Статичный SPA-прототип без сборки: `index.html`, `styles.css`, `app.js`.

## Что добавлено

- отдельные страницы через hash-router: Dashboard, Кейсы, Case Detail, Banana Upgrade, Инвентарь, Reward Hub, ЛК, Рефералка, Турниры, Ивенты;
- live feed: клик по карточке открытия ведет на страницу соответствующего кейса;
- система уведомлений пользователя: доступен кейс, доступен приз, доступно бонусное колесо, выдан промокод, результат апгрейда;
- Reward Hub со статусами бонусов и действиями: free case, Fruit Wheel, promo, daily case;
- полноценная Banana Upgrade-механика:
  - выбор 1–10 предметов;
  - входная стоимость;
  - быстрые пресеты x1.2, x1.5, x2, x3, x5, x10;
  - выбор конкретного целевого предмета;
  - расчет множителя и шанса;
  - банановые слоты с risk-state;
  - блокировка предметов во время запуска;
  - mock server result;
  - success/fail состояния;
  - быстрый Upgrade из инвентаря и из результата открытия кейса;
- страница кейсов с описанием RTP, ценности и рисков;
- личный кабинет: профиль, user ID, URL, статистика по режимам, переход в Reward Hub;
- реферальная система “Мои макаки” с иллюстративными макаками, банановым прогрессом и tier-наградами;
- турниры: отдельная страница с баннером, призами, лидербордом и special arena фоном;
- ивенты: Major Jungle Event и Banana Week с event cases;
- детальный анимированный jungle-фон, бананы, макаки, кейсы, wheel/reel/upgrade анимации.

## Модель данных

В `app.js` заведены фронтовые mock-данные:

- `cases` — кейсы, цены, RTP, risk, value score, состав;
- `targetItems` — каталог целей для апгрейда;
- `inventory` — предметы пользователя;
- `notifications` — центр уведомлений;
- `rewards` — бонусы пользователя;
- `tournaments` и `events` — турнирные и event-разделы.

Мини-бек имитируется объектом `api`:

- `api.upgrade(snapshot)` фиксирует входной пул, цель, множитель, шанс и возвращает результат;
- `api.grantReward(reward)` имитирует выдачу бонуса.

## Как открыть

Откройте `index.html` в браузере. Сборка и зависимости не нужны.
